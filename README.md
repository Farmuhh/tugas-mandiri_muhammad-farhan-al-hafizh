# tugas-mandiri_muhammad-farhan-al-hafizh
Repo ini berisi jawaban dari tugas mandiri ayng diberikan oleh PT Data Integrasi Inovasi. 

Proyek yang digarap pada tugas ini merupakan Analisis data e-commerce RetailKita, dimana saya diberikan 3 buah tabel berisi data customer, produk, dan transaksi. 
Tugas saya adalah melakukan pemmbersihan 3 tabel data tersebut sebelum menyatukannya menjadi sebuah tabel analitik tunggal untuk nantinya dianalisis.
Selain itu pada proyek ini juga saya diminta untuk mengimpor dataset yang sudah siap dianalisi ke database lokal, untuk itu pada repo ini juga saya sediakan file SQL berisikan database yang diunggah (transaction_gab_db).

Untuk menjalankan program yang telah dibuat, anda perlu memperhatikan beberapa hal sebagai berikut:
1. Untuk menjalankan python agar dataset terbaca, pastikan anda mengubah path lokasi file CSV yang anda punya. selebihnya anda bisa langsung menjalankan kode.
2. Untuk menjalankan file database yang saya telah sediakan anda perlu mengimpornya terlebih dahulu, saya menyarankan untuk menggunakan layanan phpmyadmin di XAMPP.
